# PiCam
Raspberry Pi Camera Web UI

Tested with:
- Raspberry Pi Zero W, Raspberry Pi 3B
- Raspberry Pi Camera v2
- Raspbian (32bit !)
- Python 3.9

## Setup

```shell
sudo apt install --no-install-recommends python3-picamera python3-astral python3-numpy ffmpeg
```

## Start

```shell
python picam
```
